高阶函数

高级函数是指至少满足下列条件之一的函数。

  • 函数可以作为参数被传递

    实际应用场景：回调函数（callback）、常见的数组方法，如sort filter map reduce 等

  • 函数可以作为返回值输出

    实际应用场景：闭包等各种场景
